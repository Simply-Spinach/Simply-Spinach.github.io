Code Log 2: CSS Measuring Units

In CSS, there are multiple types of units a developer can use based on what they like. For example, a developer can use units that exist in reality (like centimeters and milimeters), called absolute, or objects relative to the device they are on (such as screen size or font), called relative units.

Units are based on a coordinate grid where (0,0) is the top left of the element. X coordinates move objects to the right when positive and left when negative and Y coordinates move objects down when positive and up when negative.

Here are some examples of units you can use in css:

  • px: pixels (1/96th of an inch)
  • pt: point (1/72th of an inch)
  • em: character sizes
  • rem: root (default) character sizes
  • %: Percent of screen size

Units can be used for a variety of parameters such as

  • Position
  • Scale
  • Padding
  • Etc.
